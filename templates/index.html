Covid-19 Vaccination Rollout


The following webpage examines the early stages of the Covid vaccination rollout from a global view down to a state view. Data sets were found through Kaggle, Our World in Data, and Michigan.gov and were then converted and stored in a sql database. Using a combination of platforms including python,javascript, d3.js, c3.js, plotly, flask, and mapbox, interative visualisations were created that allow the data to be displayed in various ways.
